Having Fun with Game of Thrones Data

Git Hub
Background: The purpose of this project is to extract, transform and load the Games of Thrones data and find out the death counts of the episodes from each seasons.
Methods: This project was completed by using Python, Pandas, Postgresql. The data in csv format were found on the internet and they were merged and transformed using Pandas in Python. Then using the python script, all the data was migrated over to Postgresql.
