Machine Learning (LSTM model) Predicting Stock Prices

Website / Git Hub
Background: The purpose of this project is to create a LSTM (Long Short-Term Memoery Model) machine learning model to predict stock prices. Then publish the analysis and results of our data along with visualization on a website
Methods: This project was completed by using Python, Sklearn, Tensorflow, Numpy, Pandas, Matplotlib and API's. Multivaraible LSTM model was chosen for this project to predict future stock prices and most of the stock data such as price, ratios, market and volitality indexes were obtained from Financialmodelingprep API. All of the obtained data were extracted, transformed and loaded using Pandas and Numpy. More detailed explanations and references can be found in Git Hub and the slides below.
