Web Scraping

Git Hub
Background: The purpose of this project is to build a python module that will scrap current Mars data from the the internet and display the changes on a website when a user clicks a button. All of the data will be scrapped from the NASA Jet Propulsion Laboratory and the Astrogeology website.
Methods: This project was completed by using html and Python, along with several libraries like Splinter, Flask, Pymongo and BeautifulSoup. Flask and BeautifulSoup will allow the user to scrape the real time data when the button is clicked.
