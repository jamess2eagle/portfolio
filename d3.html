Interactive Chart

Website / Git Hub
Background: The purpose of this project is to use 2014 ACS 1-year estimates data in CSV format and build an interactive chart on a website. The dataset contains various aspects like healthcare, smoking, obesity, income, age and poverty data.
Methods: The website was built by using html, css, js along with d3 library. A user can filter and analyze the data by using a selecting axis titles.
